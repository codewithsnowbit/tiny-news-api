
# Tiny - A News API

It is a News Fetching totally *free* API (Application programming interface) for your next projects




## API Reference

#### Get all news

```http
  GET /api/news
```

#### Get Sports News

```http
  GET /api/sports
```


#### Get Climate/Environment related news

```http
    GET /api/climate
```
